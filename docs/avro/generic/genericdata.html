    Module avro.generic.genericdata

    Classes used to access generic Avro data using a schema without pre-compiled classes.

    Classes

    NameDescription
    GenericArray A generic container for Avro arrays.
    GenericContainer The base class for all generic types that act as containers.
    GenericDatum GenericDatum which can hold any Avro type. The datum has a type and a value. The type is one of the Avro data types. The D type for value corresponds to the Avro type. - An avro null corresponds to no D type. It is illegal to try to access values for null. - Avro boolean maps to D bool - Avro int maps to D int. - Avro long maps to D long. - Avro float maps to D float. - Avro double maps to D double. - Avro string maps to D string. - Avro bytes maps to D ubyte[]. - Avro fixed maps to D class GenericFixed. - Avro enum maps to D class GenericEnum. - Avro array maps to D class GenericArray. - Avro map maps to D class GenericMap. - There is no D type corresponding to Avro union. The object should have the D type corresponding to one of the constituent types of the union.
    GenericEnum A generic container for Avro enums.
    GenericFixed A generic container for Avro fixed.
    GenericMap A generic container for Avro maps.
    GenericRecord The generic container for Avro records.
    GenericUnion A generic container for unions.
