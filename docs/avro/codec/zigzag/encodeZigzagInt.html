    Function encodeZigzagInt

    Perform zig-zag encoding for a 32-bit int.

    uint encodeZigzagInt (
      int input
    ) pure;

    Example

    assert(encodeZigzagInt(0) == 0);
assert(encodeZigzagInt(-1) == 1);
